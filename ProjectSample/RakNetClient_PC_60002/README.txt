This project is IM PC client project.
To start this client should run core server first.

1.To build this project,just run make in current dictionary.
2.Client just tested on Ubuntu,it works on ubuntu 12.04
3.After build successfully the .so file achieved at ./bin.
4.Config client,just cd ./config,and modify the config file.
5.If u want to read run time log,just cd ./log
